# Deploying a Static Website with GitHub Pages

This project demonstrates how to deploy a static website using GitHub Pages and assign a custom domain.

## Steps Followed

1. **Created a GitHub repository**  
   Repository name: `deployingstatic`

2. **Added website files**  
   Included an `index.html` file in the root directory.

3. **Initialized Git and pushed files**  
   Used Git commands to track and upload files to GitHub.

4. **Enabled GitHub Pages**  
   Set the source branch to `main` and folder to `/ (root)` in repository settings.

5. **Assigned a custom domain**  
   Configured DNS records to point the domain to GitHub Pages.
